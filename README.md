# Porsche Website Project

Это статический сайт про автомобили Porsche. На сайте представлены разные модели автомобилей, их страницы с описанием, изображениями и отдельными стилями для каждой модели. Также есть JavaScript для дополнительной логики и взаимодействия на главной странице.

## Как запустить

Открой файл `index.html` в браузере.
Проект не требует установки серверов или дополнительных программ.
